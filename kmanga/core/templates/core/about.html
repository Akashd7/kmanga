KManga is a service were you can read your favorites mangas in your kindle.

We are in BETA mode, so expect problems in the service. If you find some issues you can contact us via the contact form.

Help

  1. Set in your user profile your kindle email.
  2. Allow in your amazon account to receive emails from kindle@kmanga.net.
  3. Use the search box to find your manga and subscribe to it.
  4. Configure in the subscription the language and number of issues per day.
  5. Configure in your profile your time zone and the UTC time when your subscription will be send.

The system will send you a maximum of 10 issues per day, but there is no limit of the number of subscriptions that you can create.

Check the Action select box to change the status of the issues, and to manually send a list of issues.

Source code

The source code of kmanga.net is available in GitHub.

Also, one disclaimer, kmanga do not store any manga. It only download the images from the different sources and build a MOBI document specific for the subscriber.
